# README

* Start the project

```shell
  rails s
```

* Start the console

```shell
  rails c
```

* Generate GraphQL object

```console
rails g graphql:object Post title:String
```

* Create Database

```
  rails db:create
```

* Migrate Database

```
  rails db:migrate
```
